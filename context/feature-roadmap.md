# StoryFit 기능 로드맵

## 구현된 기능

| 기능 | 비고 |
|---|---|
| 캐릭터 상세 설정 (이름·말투·성격·배경) | systemPrompt, additionalInfo, tags |
| 예시 대화 (Dialogue Examples) | exampleDialogues 필드, 시스템 프롬프트 조립에 포함 |
| 행동지문 (`*action*`) 분리 렌더링 | 이탤릭 + 회색 스타일 적용 |
| 답변 재생성 (Regenerate) | 마지막 AI 메시지 삭제 후 재생성 |
| 메시지 편집 | 유저 메시지 수정 시 이후 메시지 삭제 |
| Lorebook | 키워드 매칭, 우선순위·토큰 캡 적용 |
| 핵심 기억 / 상태 타임라인 고정 | coreMemory, statusTimeline |
| 장기 기억 자동 요약 | isSummarizing 플래그, 중복 방지 |
| 유저 페르소나 등록·선택 | 대화마다 다른 페르소나 선택 가능 |
| 소설 모드 | 작가 시점 별도 모드 존재 |
| Safety Settings 조절 | strict / standard / relaxed |
| 시스템 지침 조립 순서 고정 | 8단계 순서 엄격히 유지 |
| AI 모델 선택 | Gemini / ChatGPT / Claude 어댑터 |

## 미구현 기능

| 기능 | 설명 | 난이도 |
|---|---|---|
| **첫 메시지 분기** | 캐릭터가 대화를 먼저 시작하는 인사말을 여러 개 등록해두고, 대화방 입장 시 랜덤 or 선택해서 시작. "안녕하세요" 대신 상황마다 다른 첫 문장으로 몰입감 상승 | 낮음 |
| **스와이프 답변** | AI 답변이 마음에 안 들면 재생성 버튼으로 다른 버전을 받고, 이전/다음 버튼으로 여러 버전을 비교해서 마음에 드는 걸 선택. 현재는 재생성하면 이전 버전이 사라짐. Message.parentId 구조는 이미 준비됨 (v2 예정) | 중간 |
| **봇 복사·수정 (RemiX)** | 다른 사람이 만든 공개 캐릭터를 내 캐릭터 목록으로 복사해 와서 이름·성격·말투 등을 내 입맛대로 수정해 사용 | 중간 |
| **관계 스탯 수치화** | 대화 내용에 따라 캐릭터의 호감도·신뢰도·집착도 등 수치가 변함. "오늘 칭찬했더니 호감도 +5" 같은 방식으로 관계 진전을 눈으로 확인 | ✅ 구현 완료 (스토리 모드 한정, 스탯 태그 선택 방식) |
| **조건부 해금 시스템** | 호감도가 일정 수준 이상이 돼야 특정 대사나 시나리오가 열림. 유저가 매일 접속해서 관계를 쌓도록 유도하는 리텐션 기능 | 높음 |
| **그룹 채팅** | 캐릭터 여러 명과 유저가 한 대화방에서 동시에 대화. 예: A와 B가 서로 말하고 유저가 중재하는 형식. DB 구조는 이미 배열로 준비돼 있음 | 높음 |
| **선택지 기반 스토리 모드** | 자유 대화가 아니라 AI가 상황을 제시하고 유저가 1·2·3번 선택지를 고르며 진행하는 비주얼 노벨·게임 형식 | ✅ 구현 완료 |
| **멀티 루트 스토리 (복수 주인공)** | 스토리 생성 시 후보 캐릭터 2~4명 선택. AI가 여러 캐릭터가 동시 등장하는 장면을 쓰고, 유저의 선택에 따라 캐릭터별 스탯(호감도 등)이 독립 추적됨. 엔딩 시점에 스탯 기반으로 최종 주인공 추천 또는 직접 선택. **구현 방향:** 기존 스토리 모드 확장 (새 모드 불필요) — ① 생성 시 복수 캐릭터 선택 허용 ② 스탯을 캐릭터별로 분리(`{characterId, name, value}`) ③ AI 프롬프트에 후보 캐릭터 전원 등장 지시 ④ 엔딩 버튼 추가 | 높음 |
| **AI 보이스 콜** | 텍스트 채팅 대신 실시간 음성 통화. 유저가 말하면 캐릭터 목소리로 답변 (TTS+STT). Character.AI가 최근 가장 밀고 있는 기능 | 높음 |
| **셀카·상황 이미지 생성** | 대화 중 캐릭터가 현재 상황에 맞는 이미지를 메시지로 전송. 캐릭터 외모 설정 + 현재 상황을 조합해 AI 이미지 실시간 생성 | 높음 |
| **컨텍스트 캐싱** | 유저에게 보이는 기능은 아니고 백엔드 최적화. 긴 대화 기록을 Gemini 서버에 캐시해두면 매번 전체를 전송할 필요가 없어 API 비용 최대 80% 절감. **보류** — 현재 모델(gemini-2.5-pro 프리뷰)이 캐싱 미지원. 정식 출시 후 적용 예정 | 중간 |
