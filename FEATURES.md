## 몰입/게임성

### 1. 스탯 기반 판정 (주사위)
- 위험하거나 불확실한 행동 입력 시 주사위 + 관련 스탯으로 성공/실패 판정
- 판정 결과(대성공/성공/실패/대실패)를 시스템 프롬프트에 주입해 AI가 결과를 서사에 반영
- 기존 스탯 시스템(statsConfig) 재활용
- UI: 판정 발생 시 주사위 굴림 연출 + 결과 뱃지를 메시지에 표시

### 2. 관전 모드 (멀티스토리 자동 진행)
- 멀티스토리에서 "자동 진행 N턴" 버튼 → 사용자 입력 없이 캐릭터들끼리 대화 진행
- 기존 턴 순서(turnOrder) 로직에 루프 추가
- 진행 중 언제든 중단하고 직접 개입 가능

### 3. 장면 삽화 생성
- 현재 장면을 Gemini 이미지 생성으로 삽화화해 채팅에 삽입
- 최근 메시지 내용 → 이미지 프롬프트 변환 → 생성 → 메시지로 저장
- 명장면 기록용

## 편의성

### 4. 대화 본문 전체 검색 ★ 1순위
- 현재 chatlist 검색은 제목/캐릭터명만 가능 → 메시지 본문 검색 추가
- 키워드 검색(ILIKE) + 의미 검색(pgvector, 기존 임베딩 인프라 재활용)
- 검색 결과 클릭 시 해당 대화의 해당 메시지 위치로 이동

### 5. "지금까지의 줄거리" 회고 ★ 2순위
- 오래 쉰 대화에서 한 버튼으로 줄거리 요약 보기
- 서버의 메모리 요약(memorySummarization) 로직 재활용, UI 노출만 추가
- 위치: 채팅방 헤더 또는 사이드패널

## 감성

### 6. 메시지 북마크
- 명장면/명대사 핀 찍기 + 대화별/전체 모아보기
- Message에 bookmarked 플래그 추가

### 7. 캐릭터가 먼저 말걸기
- 마지막 메시지로부터 경과 시간을 인식해 재접속 시 캐릭터가 먼저 인사
- 예: 3일 만에 접속 → "3일 만이네...?"
- 경과 시간 + 마지막 장면 요약을 프롬프트에 넣어 자연스러운 재회 인사 생성

### 8. 완결작 소설 내보내기
- 서재의 완결작을 챕터 단위로 정리된 텍스트/EPUB으로 내보내기
- 선택지 블록 제거, 본문만 정제(기존 parseStoryChoices 재활용)
